Procesadores CSS

Es un programa que convierte un tipo de información en hojas de estilo CSS (Boostrap y Foundation están escritos sobre Sass (SCSS))

Uso de CSS

Anidación

Es posible anidar los selectores de los elementos descendentes, lo que permite visulizar mejor la jerarquía de los elementos. Debe utilizarse cuidadosamente, ya que la especificidad de los elementos puede aumentar sin darnos cuenta.

Variables

Tal como las variables CSS, cada preprocesador cuenta con la alternativa de crear variables reutilizables. Las variables se escriben "$variable: #ccc;" y deben estar declaradas (escritas) antes de ser utilizadas.

Operaciones

Otra cualidad que consideran casi todos los preprocesadores es poder realizar operaciones matemáticas dentro del valor de una propiedad. SCSS considera al menos +, -, *, /, y %.

Funciones

Sass ofrece una larga lista de funciones que pñodemos utilizar para gestionar colores, hacer cálculos o programar funciones más complejas.

Import

Una de las principales ventajas de un preprocesador es la capacidad de importar otros archivos scss (o css), lo que permite aislar estilos en hojas separadas.

  • @import "reset.scss"
  • @import "type.scss"
  • @import "other.scss"

Mixins

Forma de crear funciones propias, para grupar estilos comunes que puedan generarse a partir de variables. Podemos definir los parámetros y comportamientos internos a partir de ellos.

  • @mixin nombre ($parametros) { //estilos }
  • @include nombre ($parametros)
