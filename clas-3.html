Clase 3

Introducción a CSS

Cascading Style Sheets (Hoja de estilos en cascada)

  • Forma de escribirlo: regla
  • Selector - Propiedad - Valor de la propiedad (es una declaración)
  • Se pueden incorporar multiples selectores y múltiples declaraciones

En la escritura

  • Utilizar minúsculas
  • Identar declaraciones dentro de la regla
  • Solo un selector por línea
  • Solo una declaración por línea
  • Dejar espacio entre el último selector y la llave de inicio
  • Dejar un espacio entre los dos puntos y el valor

¿Cómo asociar una hoja de estilos a un documento HTML

Como archivo externo mediante link

"< link rel “stylesheet” href= “estilos.css” medi-= “screen” >"

  • Se puede insertar tanto en la sección "head" como en el "body", pero se recomienda en el "head"
  • Se puede enlazar a múltiples hojas de estilos
  • Las hojas de estilo se guardan en caché de navegador por lo que optimiza la descarga de la página

Etiqueta Style

  • Pueden haber múltiples delcaraciones
  • Permite usar todas las posibilidades de CSS

Estilos en línea

"h1 style "background: red; color: white;" Hola Mundo"

  • Mezcla capa de contenido con capa de presentación
  • Sólo se puede dar estilos a un elemento a la vez
  • Amentan el tamaño de la descarga del HTML ya que se descargan con éste en cada acceso
  • Al hacer una modifiación, se debe replicar en todas las páginas de un sitio

Selectores CSS

Los selectores definen sobre que elementos se aplicará el conjunto de reglas CSS

  • Etiqueta
  • Atributo id
  • Atributo class

Especificidad: es la jerarquía de aplicación de estilos sobre los elementos. Tiene un valor y se toma siempre el mayor.

  • Elementos, pseudo elementos: 1
  • Clases, pseudo clases y atributos: 10
  • Ids: 100
  • Estilos en linea: 1000
