Clase 4

Variables CSS

Entidad que guarda un valor con el fin de ser reutilizado

  • Se recomienda declarar en la raíz del documento "html :root"
  • Se utilizan a través de la función var()

Tipografías para web

"font-family"

Se puede definir por estilo Serif - Sans serif o por familia tipografía: Open Sans, Helvetica, Arial, etc. (si las fientes las tengo en mi computador se utilizan automáticamente, si no se deben descargar)

Fuentes variables

Son tipografías específicas para la web ej: Clash Grotesk "@font-face"

Tamaños y espaciados del texto

  • font-size (tamaño del texto)
  • line-height (interlineado)
  • letter-spacing (interletrado)
  • word-spacing (espacio entre palabras)

Alineación del texto

  • text-align (left)
  • text-align (center)
  • text-align (right)
  • text-align (justify)

Color del texto, como declararlo

  • yellow, red, purple, etc (por palabra clave)
  • #fffff (hexadecimal)
  • rgb(), rgba() (red, green, blue)
  • hsl(), hsla() (hue, saturation, lightness)
  • transparent/currentColor

Otras propiedades de texto

  • text-decoration: underline (subrayado)
  • text-shadow (sombra)
  • text-overflow (desbordamiento)
  • white-space (uso de blancos dentro de un elemento)

Modelo de caja

(todo es una caja)

Se utiliza la etiqueta: (Display: inline, block, inline-block)

Posiciones

  • Position: absolute
  • Position: sticky

Control de flujo

  • Rebalse del contenido - overflow
  • Tipo de posicionamiento - position (static, relative, absolute)
  • Jerarquía de capas - z-index
